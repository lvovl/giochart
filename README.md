#GrowingIO Chart component

## Install
在package.json中的dependencies里添加
```
"giochart": "git+https://github.com/yuyangvi/giochart.git#master",
```
在package.json所在目录执行
```
npm install giochart
```
## Development
```
npm install
npm start
```

## Example
http://localhost:9090
